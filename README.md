📊 Project Overview
This repository contains a comprehensive Power BI Dashboard that analyzes the relationship between rainfall patterns and agricultural output. The project transforms raw environmental and farming data into visual insights to help understand how precipitation variability influences food security and state-wise agricultural performance.
🚀 Key Features
The dashboard is organized into three primary analytical views:
1. Rainfall Impact on Agricultural Productivity
Trend Monitoring: Visualizes yearly rainfall fluctuations alongside total crop production.
Impact Correlation: Identifies how deviations from average rainfall (surplus or deficit) directly affect harvest volumes.
2. Rainfall Analytics & Crop Correlation
Crop-Specific Deep Dives: Detailed analysis of how specific crops (e.g., Rice, Wheat, Pulses) respond to rainfall.
Growth Cycle Mapping: Analyzes precipitation requirements during different sowing and harvesting stages.
3. State Performance Explorer
Regional Benchmarking: A geospatial comparison of agricultural productivity across different states.
Performance Metrics: Ranks states based on yield efficiency and rainfall resilience.
🛠️ Tools & Technologies
Power BI Desktop: Used for data modeling and visualization.
DAX (Data Analysis Expressions): Utilized for creating complex measures, such as "Year-over-Year Growth" and "Rainfall Deviation Index."
Power Query: Employed for data cleaning, ETL processes, and normalizing agricultural datasets.
📂 File Structure
Menaga.A_CPDA_Batch_10_.pbix: The core Power BI file containing all data connections, models, and dashboard pages.
